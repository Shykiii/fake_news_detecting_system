# Fake News Detection System

## 📌 Problem
Detect whether a news article is Fake or Real using NLP.

## 🧠 Approach
- Text Cleaning
- TF-IDF Feature Engineering
- Models: Naive Bayes, Logistic Regression, SVM

## 📊 Results
- Best Model: SVM
- Accuracy: 87%

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK

## 🚀 How to Run
1. Install requirements
2. Run notebook or app.py
